Sanitizador de Dados Entropia de Shannon

🧪 O que esta ferramenta faz?

Ela mede a "desordem" de cada trecho de texto usando a fórmula de Claude Shannon (1948). Textos humanos coerentes têm entropia baixa (~4.0). Lixo digital, hashes e dados corrompidos têm entropia alta (>7.5). O CROM usa exatamente essa lógica para filtrar datasets antes de treinar a IA.
